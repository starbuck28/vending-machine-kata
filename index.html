The Virtual Vending Machine

Instructions:

1. Click buttons below to insert coins into the machine. Machine will make change by default.

2. To select an item, click a numbered button corresponding to desired item.

3. To request money back, click red "Return change" button.

4. To remove change, click "Collect Change" area.

5. To collect item, click area when instructed

6. To tell the machine to accept exact change only, select "Exact change" button

7. To allow machine to make change, click "Change ok" button
